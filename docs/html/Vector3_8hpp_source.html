290 
291  hum::Vector3 is a simple class that defines a mathematical
292  vector with two coordinates (x, y and z). It can be used to
293  represent anything that has two dimensions: a size, a point,
294  a velocity, etc.
295 
296  The template parameter T is the type of the coordinates. It
297  can be any type that supports arithmetic operations (+, -, /, *)
298  and comparisons (==, !=), for example int or float.
299 
300  You generally don't have to care about the templated form (hum::Vector3<T>),
301  the most common specializations have special typedefs:
302  \li hum::Vector3<float> is hum::Vector3f
303  \li hum::Vector3<int> is hum::Vector3i
304 
305  The hum::Vector3 class has a small and simple interface, its x, y and z members
306  can be accessed directly (there are no accessors like setX(), getX()) and it
307  contains no mathematical function like dot product, cross product, length, etc.
308 
309  Usage example:
310  \code
311  hum::Vector3f v1(16.5f, 24.f, 13.f);
312  v1.x = 18.2f;
313  float y = v1.y;
314 
315  hum::Vector3f v2 = v1 * 5.f;
316  hum::Vector3f v3;
317  v3 = v1 + v2;
